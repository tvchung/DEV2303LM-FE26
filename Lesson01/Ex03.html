Hiển thị hình ảnh, liên kết

Hiển thị hình ảnh: 

        - img: để hiển thị hình ảnh
            + attr: 2 thuộc tính cơ bản
                ++ src: url
                ++ alt: nội dung
        -> url: đường dẫn đến file ảnh (jpg, png, ...)
            + Đường dẫn tuyệt đối: 
                [giao thức]:// domain / [pathfolder]/ pathFile
                https://devmaster.edu.vn/images/logo.png 
            + Đường dẫn tương đối: tính từ vị trí lưu file html 
                folder/file 
                ví dụ: ex03.html
                    images/logo.png

                ví dụ: từ file page1.html 
                    hiển thị hình ảnh => đường dẫn tương đối:
                        ../images/logo.png
